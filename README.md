# Hyprland + Noctalia Shell

A collection of my personal configuration files for a sleek, Wayland-based desktop environment. Optimized for productivity and a dark, modern aesthetic.

-----

## 🖥️ System Overview

| Component | Choice |
| :--- | :--- |
| **Compositor** | [Hyprland](https://hyprland.org/) |
| **Shell & UI** | [Noctalia-shell](https://docs.noctalia.dev/) |
| **Terminal** | Kitty |
| **Web Browser** | Firefox |
| **File Manager** | Nautilus |
| **Cursor Theme** | Bibata Modern |
| **Auth Agent** | Polkit-gnome |

-----

## 📦 Dependencies

  * `hyprland-git` (or stable)
  * `noctalia-shell`
  * `kitty`
  * `nautilus`
  * `bibata-cursor-theme`
  * `polkit-gnome`
  * `xdg-desktop-portal-hyprland`
  * `ttf-iosevka-term`

-----

## 🚀 Installation


```bash
git clone https://github.com/Ametic/hypr-dotfiles.git ~/.config
```

-----

## ⌨️ Keybindings (Highlights)

| Keybind | Action |
| :--- | :--- |
| `Super` + `RETURN` | Launch **Kitty** |
| `Super` + `E` | Open **Nautilus** |
| `Super` + `W` | Launch **Firefox** |
| `Super` + `Q` | Kill Active Window |
| `Super` + `CTRL` + `P` | Power Menu |
